# Set Map Demo Project

### Description

This program demonstrates working with Sets and Maps in java. It utilizes a Hast set/map, a Tree set/map, and a Linked set/map.  

### To Run

Using terminal on Mac:

```
$ cd Set-Map-Demo
$ javac SetMapDemo.java
$ java SetMapDemo
```
